## Как добавлять фото
1. Зайди в нужную папку на GitHub
2. Нажми Add file → Upload files
3. Назови фото: 1.jpg, 2.jpg, 3.jpg ... (до 6 штук)
4. Commit — фото сразу появятся в приложении
